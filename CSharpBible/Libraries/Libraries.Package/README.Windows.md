# CSharpBible.Libraries.Windows

Dieses NuGet-Paket bündelt die Windows-spezifischen Bibliotheken aus `CSharpBible\Libraries`.

## Enthaltene Bibliotheken
- CommonDialogs
- CommonDialogs_net
- ConsoleLib_net
- ExtendedConsole
- MVVM_BaseLib
- WFSystem.Windows.Data

## Hinweise
- Das Paket enthält die kompilierten Bibliotheksartefakte für Windows-spezifische Target Frameworks.
- OS-neutrale Bibliotheken liegen im Paket `CSharpBible.Libraries.Core`.
