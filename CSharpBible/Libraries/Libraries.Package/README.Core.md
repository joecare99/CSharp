# CSharpBible.Libraries.Core

Dieses NuGet-Paket bündelt die plattformneutralen Bibliotheken aus `CSharpBible\Libraries`.

## Enthaltene Bibliotheken
- BaseLib
- CommonDialogs.Abstractions
- ConsoleDisplay
- ConsoleLib
- GenInterfaces
- MathLibrary

## Hinweise
- Das Paket enthält die kompilierten Bibliotheksartefakte für die unterstützten Target Frameworks.
- Windows-spezifische Bibliotheken liegen im Paket `CSharpBible.Libraries.Windows`.
