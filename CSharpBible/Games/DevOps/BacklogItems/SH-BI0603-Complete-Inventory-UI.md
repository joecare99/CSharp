# SH-BI0603 - Complete Inventory UI

## Work Item Type
Backlog Item

## Parent
`SH-F06 - UI and Player Experience`

## Description
Allow players to manage inventory manually through console and selected WPF UI.

## Scope
- Add inventory selection.
- Support equip, drop, inspect, and use commands.
- Show item effects and quantities.

## Acceptance Criteria
- Players can manage inventory without auto-equip.
- Inventory actions are available through keyboard and WPF commands.
- Tests cover ViewModel command behavior.
