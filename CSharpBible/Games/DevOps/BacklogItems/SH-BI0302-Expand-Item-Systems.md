# SH-BI0302 - Expand Item Systems

## Work Item Type
Backlog Item

## Parent
`SH-F03 - Content and Progression`

## Description
Expand inventory and item gameplay beyond automatic pickup and equipment.

## Scope
- Add explicit equip, unequip, drop, inspect, and pickup actions.
- Add consumables such as healing potions.
- Define stack and container behavior.

## Acceptance Criteria
- Inventory operations update map and creature state consistently.
- Equipment changes affect combat stats.
- Consumables apply effects and are removed or decremented.
- Tests cover inventory edge cases.
