# SH-BI0102 - Define Victory and Dungeon Depth Rules

## Work Item Type
Backlog Item

## Parent
`SH-F01 - Core Game Loop Completion`

## Description
Define the first complete-game objective and dungeon-depth progression rules.

## Scope
- Decide target dungeon depth.
- Add the final objective: obtaining the Amulet of JoCarneer.
- Add scoring or completion summary data.

## Product Decision
- The final victory condition is obtaining the Amulet of JoCarneer.
- Dungeon depth, boss/finale structure, and exit behavior should support the Amulet objective instead of replacing it.

## Acceptance Criteria
- A run documents the Amulet of JoCarneer as the completion objective.
- Obtaining the Amulet of JoCarneer triggers victory.
- UI-facing layers can show a victory summary.
