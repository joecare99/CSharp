# SH-T030303 - Integrate Scaling Into Spawning

## Work Item Type
Task

## Parent
`SH-BI0303 - Add Level Scaling and Rewards`

## Goal
Use depth scaling rules during enemy, item, and feature spawning.

## Scope
- Pass current depth into spawn selection.
- Apply count and quality scaling.
- Keep deterministic behavior under controlled random input.

## Done
- Deeper levels spawn different or stronger content according to rules.
- Existing level transition behavior remains valid.
