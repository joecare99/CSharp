# SH-T060401 - Select 3D Rendering Stack

## Work Item Type
Task

## Parent
`SH-BI0604 - Implement Initial 3D View`

## Goal
Select the concrete rendering stack for the first SharpHack 3D view.

## Scope
- Start evaluation with OpenGL as the preferred direction.
- Compare practical .NET-compatible options such as OpenTK, Silk.NET, Veldrid, or suitable WPF-hosted alternatives.
- Consider Visual Studio/.NET support, WPF hosting, asset pipeline, maintainability, input handling, and future prefab/mesh mapping.
- Record the selected stack and trade-offs in DevOps before implementation.

## Done
- A concrete 3D rendering stack is selected and documented.
- The decision explains why OpenGL or an alternative was chosen.
- Follow-up implementation tasks can target the selected stack without ambiguity.
