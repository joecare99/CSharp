# SH-T040203 - Add Enemy Perception Rules

## Work Item Type
Task

## Parent
`SH-BI0402 - Upgrade Enemy AI`

## Goal
Prevent enemies from always knowing the player's position unless rules allow it.

## Scope
- Add perception based on visibility, range, or sound.
- Define memory of last known player position if needed.
- Keep rules simple for the first iteration.

## Done
- AI behavior differs for seen and unseen player states.
- Tests cover visible and unseen player scenarios.
