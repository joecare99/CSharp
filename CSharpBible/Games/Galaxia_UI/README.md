# Galaxia.UI

WPF Demo-UI für das Galaxia-Projekt.

Funktionen:
- Initialisiert die Spiel-Space-Struktur.
- Listet Sektoren und deren Starsysteme.
- Beispiel-Button zum Einschiffen einer Test-Flotte (Embark).
- Platzhalter für weitere Aktionen (Double Jump).

Tests:
- MainViewModelTests prüfen grundlegende ViewModel-Logik (Initialisierung / Statusänderung).
