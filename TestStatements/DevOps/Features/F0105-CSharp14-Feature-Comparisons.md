# F0105 - C# 14 Feature Comparison Project

## Parent Epic

- E0100 - Evolve TestStatements as a Discoverable Learning Platform

## Summary

Plan a dedicated `TestStatements.CS14` project that demonstrates new C# 14 language features and contrasts them with equivalent or approximate non-C# 14 alternatives.

## Value

- Makes new language features easier to understand through side-by-side comparison
- Preserves educational value for readers who do not yet use the newest compiler features
- Expands the repository with a focused language-evolution learning path
- Creates a practical comparison surface between modern and more traditional implementations

## Scope

- A dedicated `TestStatements.CS14` project concept
- Comparison examples for new C# 14 features
- Explicit non-C# 14 alternatives or fallback implementations
- Documentation of feature value, readability impact, and migration relevance

## Backlog Candidates

- B0111 - Define the TestStatements.CS14 project structure
- B0112 - Define C# 14 feature comparison scenarios
- B0113 - Define documentation and expected-output strategy for C# 14 comparisons

## Assumptions

- The project should complement the existing `TestStatements` family rather than replace older example projects.
- Comparisons should focus on educational clarity, not only novelty.
- Non-C# 14 alternatives may be approximate when no exact older-language equivalent exists.
