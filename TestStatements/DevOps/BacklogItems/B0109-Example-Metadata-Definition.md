# B0109 - Define Reusable Metadata for Examples

## Parent Feature

- F0104 - Example Metadata Foundation

## Summary

Define the shared metadata fields that examples should expose so documentation, launchers, UI views, and validation tasks can rely on a common description model.

## Acceptance Criteria

- Required metadata fields are listed.
- Optional metadata fields are listed.
- The purpose of each field is described.
- The metadata design is reusable outside a single UI scenario.

## Child Tasks

- T0118 - Define the baseline metadata shape
- T0119 - Define metadata usage rules and ownership
