# B0101 - Document Expected Output for Learning-Critical Examples

## Parent Feature

- F0100 - Example Catalog and Learning Map

## Summary

Document the expected output or observable behavior of examples where the result is part of the teaching value.

## In Scope

- Console output examples
- Reflection and diagnostics examples
- Async sequencing examples
- Serialization and loader examples

## Acceptance Criteria

- Output-sensitive example groups are identified.
- Expected output is described in clear text.
- Non-deterministic behavior is marked as such.
- Cross-framework differences are noted when important.

## Child Tasks

- T0102 - Identify output-sensitive examples
- T0103 - Draft expected-output notes for advanced examples
