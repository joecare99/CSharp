# TestGJKAlg

WinForms application (net9.0-windows) demonstrating or visualizing the GJK (Gilbert–Johnson–Keerthi) collision detection algorithm.

## Focus
- Real-time geometric visualization
- Interactive shape manipulation (expected)
- Educational insight into computational geometry

## Dependencies
Microsoft.CSharp, DataSetExtensions (legacy helpers)

## Build
`dotnet build TestGJKAlg/TestGJKAlg.csproj`
