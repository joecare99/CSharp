# Task: Übersetzungs-Konfigurationsoberfläche dokumentieren

## Ziel

Die WinGUI-Übersetzungsoberfläche soll als eigenständiger administrativer Pflegepunkt dokumentiert werden.

## Inhalt

- Eingelesen werden `CtrlConfigTranslation` und die zugehörigen Translation-APIs.
- Dokumentiert werden Suche, Bearbeitung, Import, Export und Synchronisation.
- Zusätzlich werden die sichtbaren Abhängigkeiten zu Sprache, Term-Enums und Datenbanktabellen festgehalten.

## Ergebnis

- Feature-Dokument angelegt
- Backlog für Verbesserungen abgeleitet
- Fortschrittspunkt aktualisiert
