# 🚀 Projektübersicht: C# LLM IDE (Scrum-Ansatz)
## 🎯 Vision
Eine plattformunabhängige (OS-agnostische) C#-Entwicklungsumgebung unter Linux, die direkt mit lokalen oder entfernten Large Language Models interagieren kann.

## 🛠️ Technologie-Stack
| Komponente | Technologie | Zweck |
|-|-|-|
| Frontend/UI | Avalonia UI | Die plattformübergreifende Benutzeroberfläche. |
| Backend/Logik | C# (.NET) | Die gesamte IDE-Logik und die LLM-Interaktion. |
| Datenbindung | MVVM (Model-View-ViewModel) | Für saubere und testbare Architektur. |
| LLM-Kommunikation | HTTP / gRPC | Verbindung zu externen oder lokalen LLM-Servern. |
| Plattform | Linux (Zielplattform) | Betriebsumgebung. |
