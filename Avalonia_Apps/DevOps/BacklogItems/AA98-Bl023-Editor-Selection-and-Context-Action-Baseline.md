## Scope
Define the first editor selection and context action baseline for `AA98_AvlnCodeStudio` so later AI-assisted actions can operate on explicit user-selected content and document context under clear boundaries.

## Goals
- Define the baseline model for selection and context actions.
- Clarify which editor and workspace context may be exposed.
- Align the model with explicit user-triggered workflows.
- Keep the baseline compatible with privacy and consent principles.

## Assumptions
- Early AI actions should be explicit and user-triggered.
- Selection- and file-oriented actions are safer early directions than broad automation.
- Broad autonomous context gathering belongs to later increments.

## Open Questions
- Which context scopes should exist from the first version onward?
- How should context boundaries align with future consent levels?
